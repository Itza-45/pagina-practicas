HEAD


Funcion: Provee información general (metadatos) acerca del documento, incluyendo su título y enlaces a scripts y hojas de estilos.

Atributos Globales:Accesskey, autocapitalize, class, contextmenu, data, dir, draggable, hidden, id, inputmode, style, title, translate, por mencionar algunos.

Sintaxis 

		<-html>
           <-head>
    
           <-/head>
         <-/html>

Ejemplo 
		 <-html>
           <-head>
            <-title>Mi primera página HTML<-/title>
           <-/head>
         <-/html>

BODY


Funcion: Representa el contenido de un documento HTML. Solo puede haber un elemento body en un documento.

Atributos Globales:Alink, background, bgcolor, bottommargin, leftmargin, link, rightmargin, text, topmargin, vlink, por mencionar algunos.

Sintaxis 

		 <-html>
           <-body>
    
           <-/body>
         <-/html>

Ejemplo 
		 <-html>
           <-head>
            <-title>Titulo del documento<-/title>
           <-/head>
		   <-body>
		     <-p>Este es un parrafo<-/p>
			<-/body>
         <-/html>

SOURCE


Funcion: Especifica recursos de medios múltiples para los elementos <-picture>, <-audio>, o <-video>. Es un elemento vacío. Normalmente se utiliza para servir el mismo contenido multimedia en varios formatos soportados por diferentes navegadores.

Atributos
Sizes, src, type, media.

Sintaxis 

	     <-video controls>
         <-source src="foo.webm" type="video/webm">
         <-source src="foo.ogg" type="video/ogg">
         <-source src="foo.mov" type="video/quicktime">
         <-/video>

Ejemplo 
	    <-audio controls>
        <-source src="/assets/audio/Jahzzar_The_Flowers_Are_Still_Standing.ogg" 
		type="audio/ogg">
        <-/audio>

PRE


Funcion: Representa texto preformateado. El texto en este elemento típicamente se muestra en una fuente fija, no proporcional, exactamente como es mostrado en el archivo. Los espacios dentro de este elemento también son mostrados como están escritos.

Atributos
Globales:Accesskey, autocapitalize, class, dir.

Sintaxis 

	     <-body>
         <-pre>
         color:  red;
         ---------------------------------
         color:green;
         <-/pre>
         <-/body>

Ejemplo 
	     <-pre>
         …… 
         ---------
         ////////
         ))))((((
         <-/pre>

         <-head>...<-/head>

         <-/html>

         <-title>Mi página de ejemplo<-/title>
